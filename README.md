El Mapa del Tesoro Vocacional

Página web interactiva basada en la idea: el tesoro no es una carrera específica, sino el descubrimiento de tus talentos e intereses.

Estructura del proyecto

mapa-tesoro/
├── index.html   → estructura y contenido
├── styles.css   → estilos (paleta "mapa antiguo", tipografía, layout)
└── script.js    → interactividad (excavar coordenadas, abrir el cofre)

Cómo funciona

La página presenta un viaje de 4 estaciones (coordenadas en el mapa). En cada estación hay 3 preguntas:


2 preguntas de opción múltiple, donde cada respuesta suma un punto a una categoría (arte, tecnología, social, ciencia, naturaleza, lógica, negocios).
1 pregunta abierta de reflexión, que se guarda y se muestra de vuelta al final como parte del recorrido.


Al terminar las 4 estaciones se abre el cofre del tesoro y se calcula la categoría con más puntos. El resultado no es el nombre de una carrera, sino una descripción del perfil de talentos e intereses de la persona (por ejemplo "Creador/a", "Investigador/a curioso/a", "Conector/a de personas"), junto con una barra visual de cómo se distribuyeron sus puntos entre categorías y un resumen de sus respuestas de reflexión.
